# Aurum_Noctis
Doberman Kennel
A modern, responsive web presentation for a fictional Doberman breeding kennel. This project serves as a showcase of my frontend development and web design skills.

🔗 Live Demo
View Live Website here

🚀 Key Features
Fully Responsive: Optimized for desktop, tablets, and mobile devices.

Dynamic Gallery: Interactive photo showcase of the dogs.

Contact System: Integrated form and social media links.

Clean UI/UX: Dark-themed professional aesthetic tailored to the breed's character.
